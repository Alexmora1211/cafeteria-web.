# ☕ CafeEDEC Express

## Proyecto Académico

**CafeEDEC Express** es una aplicación web desarrollada como proyecto académico para mejorar el servicio de la cafetería del Campus EDEC. El sistema permite consultar el menú, realizar pedidos en línea y reducir los tiempos de espera de los estudiantes.

---

# Objetivo General

Desarrollar una plataforma web que facilite la compra de alimentos dentro de la cafetería escolar mediante pedidos en línea.

---

# Objetivos Específicos

- Consultar el menú del día.
- Visualizar imágenes y precios de los productos.
- Realizar pedidos desde cualquier dispositivo.
- Disminuir las filas en la cafetería.
- Mejorar la experiencia del estudiante.
- Facilitar la administración del menú.

---

# Funcionalidades del Sistema

- Consulta del menú.
- Búsqueda de productos.
- Carrito de compras.
- Registro de pedidos.
- Pago del pedido.
- Consulta del estado del pedido.
- Administración del menú.

# Metodología

Para el desarrollo del proyecto se utilizó la metodología ágil **SCRUM**, organizando las actividades en sprints y utilizando historias de usuario para definir las funcionalidades del sistema.
